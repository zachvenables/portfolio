--------
Lab1
--------

This is a simple game in which a 1st person object moves around to touch a moving sphere object.


---------
Controls
---------

Up Arrow: Forward
Down Arrow: Backwards
Left Arrow: Turn left
Right Arrow: Turn Right


----------
Objective
----------

The objective is to move to the moving sphere object.  Once you make contact with the sphere, 
you will be notified that you win.


------------------
Starting the game
------------------

No install is necessary, just double-click the Lab1.exe file and the game will begin once it is loaded.


----------------
Ending the game
----------------

Once you have completed the objective, you can press alt + f4 to close the game.
